# Session: n2x2-2 — Refactor 2x2 Solver Delegation

Branch: `n2x2-2`
Date: 2026-03-05
Parent sessions: `n2x2-1` (initial 2x2 support), `n2x2-3-beginner-skeleton` (beginner solver skeleton + model de-cheating)

---

## Overview

Refactored how the codebase delegates to the 2x2 solver. Previously, the `Solvers` factory intercepted at construction time (`by_name()` checked `cube.size == 2` and returned a 2x2 solver instead of the requested one). Now delegation happens at **solve-time** inside `AbstractSolver.solve()`, which is architecturally cleaner — the solver itself discovers it's on a 2x2 cube and delegates.

### 5. Updated `Solvers` factory

- Added `default_2x2(op)` — reads config, looks up `SolverName`, calls `by_name()`
- Removed size==2 interception from `by_name()` — factory now just creates whatever is requested
- Added `TWO_BY_TWO_IDA` and `TWO_BY_TWO_BEGINNER` cases to the match statement

### 6. Added 2x2 delegation in `AbstractSolver`

**Delegated methods** (when `cube.size == 2` and `not self._is_2x2_solver`):

| Public method | Internal override | Notes |
|---|---|---|
| `solve()` | `_solve_impl()` | Delegates before any solve logic |
| `status` | `_status_impl` | Property, abstract in subclasses |
| `supported_steps()` | `_supported_steps_impl()` | Method, abstract in subclasses |
| `diagnostic()` | `_diagnostic_impl()` | Method, default prints "no diagnostics" |
| `solution()` | — | Auto-delegates (calls `solve()` internally) |

**Not delegated** (no need):
- `get_code` — should show original solver name (e.g. "Kociemba")
- `is_solved` — just checks `cube.solved`
- `get_block_statistics()` — no blocks for 2x2

The 2x2 delegate solver is **cached** in `_2x2_delegate_cache` (created lazily on first access).

### 7. Renamed methods in all concrete solvers

- `status` → `_status_impl` (10 files)
- `supported_steps()` → `_supported_steps_impl()` (8 files)
- `diagnostic()` → `_diagnostic_impl()` (1 file: DirectLayerByLayerNxNSolver)

## Architecture After This Session

```
AbstractSolver.solve()          ← @final template method
    if cube.size == 2 and not _is_2x2_solver:
        delegate to Solvers.default_2x2(op)   ← reads config
    else:
        _solve_impl(what)                      ← normal path

AbstractSolver.status           ← concrete, delegates or calls _status_impl
AbstractSolver.supported_steps  ← concrete, delegates or calls _supported_steps_impl
AbstractSolver.diagnostic       ← concrete, delegates or calls _diagnostic_impl

Solver2x2Base(BaseSolver)       ← _is_2x2_solver = True
    ├── Solver2x2IDA            ← IDA* optimal (≤11 moves)
    └── Solver2x2Beginner       ← Beginner LBL (stubs, NotImplementedError)
```
