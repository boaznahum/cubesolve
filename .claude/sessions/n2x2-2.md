## Algorithm Summary
1. **Pre-orient cube** so DBL corner is at slot 7 with co=0 (24-entry lookup table)
2. **Extract coordinates**: twist (0-728) + permutation (0-5039)
3. **IDA* search** with exact pruning table (3.5MB BFS from solved)
4. **Play moves**, then **fix virtual centers** (2x2 domain model quirk)
